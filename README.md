# Проект: Путешествие по России

### О проекте
Одностраничный сайт о путешествиях по России. Содержит основную информацию о различных направлений путешествия по России, и ссылки на полезные ресурсы.

**Технологии:**
HTML
* Семантические теги

CSS
* Flexbox
* Grid-layout
* Позиционирование
* Адаптивная верстка
* БЭМ - nested
* Псевдоклассы CSS

Дальнешее улучшение:
* Добавить Java-script
